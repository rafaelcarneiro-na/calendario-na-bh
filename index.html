🔥 Configurar Firebase

Conecte ao Firebase Realtime Database para sincronização em tempo real entre todos os usuários.

Passo 1

Acesse console.firebase.google.com → Crie um projeto → Vá em Build → Realtime Database → Crie banco em modo teste.

Passo 2

Copie a URL do banco (ex: https://meu-projeto-default-rtdb.firebaseio.com).

Passo 3

Cole abaixo e clique Conectar. Todos que usarem a mesma URL verão as mesmas edições.
